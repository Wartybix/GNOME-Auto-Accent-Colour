# Auto Accent Colour
Automatically set the GNOME accent colour based on the user's background.

## Development
Get the source:
```
git clone https://codeberg.org/Wartybix/GNOME-Auto-Accent-Colour/
```

To install locally, run `install.sh`. For changes to apply, log out and back in again.
Alternatively, run `test.sh`, which will install the application and open it in an embedded GNOME shell.
